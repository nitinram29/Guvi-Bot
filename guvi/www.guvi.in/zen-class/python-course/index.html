Live Classes

Python - Module 1

An Introduction to Python & Basics

What can Python do?, Why Python?, Python Syntax compared to other programming languages, Python Install, The print statement, Comments, Python Data Structures & Data Types, String Operations in Python, Simple Input & Output, Simple Output Formatting

 

Python - Module 2

Python Operators & Program Flow

Operators in python, Indentation, The If statements, examples for if, The while loop, for loop, range statement, Break & Continue

 

Python - Module 3

Functions & Modules

Create your own functions, Functions Parameters, Variable Arguments, Scope of a Function, Function Documentations, Lambda Functions & map, Create a Module, Standard Modules

 

Python - Module 4

Exceptions Handling

Errors, Exception handling with try, handling Multiple Exceptions, Writing your own Exception

 

Python - Module 5
  • File Handling

File handling Modes, Reading Files, Writing & Appending to Files, Handling File Exceptions

 

Python - Module 6

Classes In Python

New Style Classes, Creating Classes, Instance Methods, Inheritance, Polymorphism, Exception Classes & Custom Exceptions

 

Python - Module 7

Generators and iterators

Iterators, Generators, The Functions any and all, With Statement, Data Compression

 

Python - Module 8

Data Structures and Regular Expressions

List Comprehensions, Nested List Comprehensions, Dictionary Comprehensions, Specialized Sorts, Split, Working with special characters, date, emails, Match and find all, Search method

Python - Module 9

Collections

namedtuple(), deque, ChainMap, Counter, OrderedDict, defaultdict, UserDict, UserList, UserString

 

Python - Module 10

Writing GUIs in Python

Introduction, Components and Events, An Example GUI, The root Component, Adding a Button, Entry Widgets, Text Widgets, Check buttons

 

Python - Module 11

Python SQL Database Access

Introduction, Installation, DB Connection, Creating DB Table, INSERT, READ, UPDATE, DELETE operations, handling Errors

 

Python - Module 12

Regular Expressions

Split, Working with special characters, date, emails, Match and find all, Search method
